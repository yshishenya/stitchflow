# Launch Kit

This file is for communication, distribution, and launch messaging.

## Core positioning

Short version:

`StitchFlow` turns Stitch SDK into a reusable skill for coding agents.

Expanded version:

`StitchFlow` lets you generate and iterate UI with natural language, save HTML and screenshots locally, and reuse the same workflow across Codex, Claude Code, and OpenClaw.

## Why people should care

- raw SDKs are powerful but not always packaged for daily workflows
- designers and engineers both need faster UI exploration
- most teams want artifacts they can save, diff, and build from
- cross-agent compatibility reduces lock-in

## Who this is for

- solo builders shipping product ideas quickly
- design engineers who want faster iteration loops
- product teams exploring multiple UI directions
- internal tools teams that need useful screens fast

## Viral checklist

- show prompt -> screen -> HTML, not architecture diagrams
- post 3 strong screenshots, not 1
- keep the README promise simple
- give copy-paste demo commands
- make installation frictionless
- add a license before broad outreach
